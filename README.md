# PetJoy Super Store Website

A professional e-commerce website for a grocery delivery service built with HTML5, CSS3, and vanilla JavaScript.

## Project Structure

```
PETJOY2 WEBSITE/
├── index.html                 # Home page
├── pages/
│   ├── shop.html             # Shop/Products page
│   ├── about.html            # About us page
│   └── contact.html          # Contact page
├── assets/
│   ├── css/
│   │   └── style.css         # Main stylesheet (2075 lines)
│   ├── js/
│   │   └── main.js           # JavaScript functionality (462 lines)
│   ├── images/               # Image folder (for future use)
│   └── documents/            # Documents folder (for future use)
└── README.md                 # This file
```

## Features

### Home Page (index.html)
- Hero section with call-to-action
- Feature highlights
- Featured products showcase
- Special promotions section
- Category browsing
- Newsletter subscription
- Footer with quick links

### Shop Page (pages/shop.html)
- Product grid display
- Advanced filtering by category, price range, and availability
- Search functionality
- Multiple sort options (price, name, newest)
- Shopping cart with quantity management
- Mobile-friendly filter sidebar
- Scroll-to-top button

### About Page (pages/about.html)
- Company story
- Mission and vision statements
- Core values
- Call-to-action button

### Contact Page (pages/contact.html)
- Contact information cards
- Contact form
- FAQ section with toggle functionality
- Social media links
- Office location map
- Form submission handling

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Responsive design, flexbox, grid, animations
- **Vanilla JavaScript**: No frameworks or libraries
- **LocalStorage**: For shopping cart persistence

## Folder Organization Benefits

1. **Assets folder**: Keeps all static files organized
2. **CSS subfolder**: Easy stylesheet management
3. **JS subfolder**: Organized JavaScript files
4. **Images folder**: Ready for product/promotional images
5. **Pages folder**: Separates page files for better organization
6. **Root index.html**: Home page at website root for better SEO

## How to Use

1. Open `index.html` in a web browser to start
2. Navigate between pages using the menu
3. Use the shop page to browse and filter products
4. Add items to cart and manage quantities
5. Fill out contact form for inquiries
